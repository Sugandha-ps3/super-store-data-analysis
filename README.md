# Superstore Data Analysis

## 📊 Objective
Analyze sales data to identify key insights.

## 🔧 Tools Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## 📈 Analysis Performed
- Data Cleaning
- Total Sales Calculation
- Top 5 Products by Sales
- Monthly Sales Trend
- Visualizations

## 📌 Key Insights
- Strong overall sales performance
- Few products contribute most revenue
- Sales vary across months
- Category-wise differences observed
